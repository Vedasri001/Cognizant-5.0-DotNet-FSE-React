# Exercise 7 - Financial Forecasting

## Objective
Predict the future value of an investment using recursion.

## Technologies Used
- C#
- .NET Console Application

## Concepts
- Recursion
- Mathematical Computation
- Compound Growth

## Formula

Future Value = Present Value × (1 + Rate)^Years

## Sample Output

Present Value : 10000
Growth Rate   : 10%
Years         : 5
Future Value  : 16105.10
